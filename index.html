Data Basics

The tutorial I selected was data basics. It discusses how information is stored on computers and relates it to a switch being turned on and off. Code is measured in bits and bytes and the biggest measurement I recall is that 8 bits equal one byte. We use binary when talking about code because it can either be "on" or "off". I also learned that character encoding is when computers store/represent text by assinging numbers to characters. So, when there are images saved on a computer, they are encoded using binary in order for them to be saved in a file and for the computer to render the image properly (with the correct color, size, etc)
